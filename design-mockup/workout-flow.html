플로우 검증 포인트

· ex-chip: 완료 운동은 dim + ✓, 현재 운동은 흰 pill (빛이 현재에 머문다), 다가올 운동은 bone-outline
· sets grid: 완료된 세트는 dim, 현재 입력 행은 흰 outline + 흰 숫자. 다음 행은 ghost
· set check: 완료 체크는 순백 사각형 (의지의 단일 표현)
· rest timer: 큰 Archivo Black 숫자. 바뀌지만 flashing·pulsing 없음. 담담
· volume total: 세션 후 Archivo Black 56px "2.4 t" — 과거의 축적과 동일한 서체로 일관

색 정책 재확인

이 플로우에 감정색 없음. 세트 입력 칸이 빨강이 되는 순간은 없음. 모든 상태는 흑 × 흰의 강약 + 아웃라인으로 표현. 그라파이트 원칙 일관.

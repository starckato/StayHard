카드 펼침 → 슬롯 탭 → sub-sheet. 사진·카테고리 입력. ← 기록 탭

카테고리 색 정책

· 클린 · 일반 · 치팅: 선택 시 흰 1px outline + surface-2 배경. 감정색 없음.
· 금지: 유일하게 선택 시 oxblood outline. 점수가 실제로 깎이는 유일한 카테고리 → 색을 쓴다는 사실 자체가 "여기서부턴 결과가 다르다"는 신호.
· 저장 버튼까지 oxblood (금지 기록 · −30 pt) — 실행 전 마지막으로 의식적 확인.

사진 UX

dashed box = 빈 슬롯. 첨부된 사진은 checker 패턴 placeholder + 우상단 ×. 2슬롯까지만 — 식단은 기록이지 전시가 아님.

간식·음료·술

카테고리 그리드 5개. 선택된 셀만 counter 영역에 노출 (현재 간식만 활성). 수량 카운터는 + 버튼만 흰 (의지가 숫자를 키운다), − 는 outline. "저장" 후 inline 식단 카드의 간식 row에 "2개 · 커피" 요약으로 반영.
